🎓 Student Management System

A simple Student Management System built with Python using Object-Oriented Programming (OOP).
This project allows users to manage student records such as adding, viewing, searching, and deleting students.

It is created for learning and practicing Python OOP concepts.

🚀 Features

➕ Add new student

📋 View all students

🔍 Search student by ID

❌ Delete student

🧠 Simple command-line interface

🧩 Built using Object-Oriented Programming concepts

🛠️ Technologies Used

Python 3

Object-Oriented Programming (OOP)

Command Line Interface (CLI)
